# Vagrant command

#### To start vagrant 
```
vagrant up 
```
#### To stop vagrant
```
vagrant suspend
```
#### To resume vagrant
```
vagrant resume
```
#### To provison vagrant
```
vagrant provision
```
#### SSH to vagrant
```
vagrant ssh
```
#### To reload vagrant provision
```
vagrant reload --provison
```
#### List vagrant box
```
vagrant box list
```
#### Remove box 
```
vagrant box remove <box-name>
```
